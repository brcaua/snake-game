## Prerequisites

- Ter um computador Windows, Linux ou Mac

## Install

Clonando o projeto para um repositório local
```sh
git clone https://github.com/brcaua/snake-game.git
```
Acessar a pasta do projeto
```sh
cd snake-game
```
Abrir com Visual Studio Code
```sh
code .
```
